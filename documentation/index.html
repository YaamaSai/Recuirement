RecruitPro Documentation

Welcome to the documentation for RecruitPro. This guide will help you install, customize, and deploy your new recruitment platform template.

1. Installation Guide

Simply copy the entire folder structure to your web server or open pages/index.html directly in your browser.

2. Customization Guide

All colors, spacing, and typography are controlled by CSS variables in assets/css/style.css. To change the primary brand color, modify the --primary-color variable under the :root selector.

3. Dark Mode & RTL

Dark mode automatically follows system preferences, and can be toggled manually via the switch in the navbar. RTL support is available in rtl.css.

4. Support

If you have any questions, please reach out to our support team.
