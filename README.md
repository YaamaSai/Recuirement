# RecruitPro - Staffing & Recruitment Template

RecruitPro is a premium, highly responsive HTML template designed for staffing agencies, recruitment firms, and job boards.

## Features

- **Responsive Design**: Mobile-first architecture.
- **Dark/Light Mode**: Full theme switching with system detection.
- **RTL Support**: Built-in right-to-left layout compatibility.
- **Dashboards**: Dedicated portals for Candidates and Employers.
- **Premium Aesthetics**: iOS typography, skeleton loaders, and modern hover effects.
